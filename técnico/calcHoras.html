Meus pensamentos sobre esta atividade:

Habilidades Desenvolvidas: Compreensão dos conceitos básicos de HTML, JavaScript, CSS e React. Habilidade de programação em JavaScript para criar funcionalidades interativas. Aprendizado sobre a estruturação e o design de uma interface de usuário utilizando HTML e CSS. Introdução ao uso do Git e GitHub para controle de versão e colaboração em projetos de código aberto.

Objetivo da Atividade: O objetivo principal da atividade era desenvolver uma calculadora de horas utilizando tecnologias web modernas, incluindo HTML, JavaScript, CSS e React. A atividade visava ajudar os alunos a consolidar seus conhecimentos de programação e proporcionar uma introdução prática ao desenvolvimento web.
Assunto: O assunto da atividade foi a criação de uma calculadora de horas, uma aplicação útil que permite aos usuários calcular o tempo decorrido entre duas datas e horas específicas. A atividade envolveu a aplicação de lógica de programação para manipular datas e horas, bem como a criação de uma interface de usuário intuitiva para interação do usuário.

Crítica Pessoal: Como aluno do ensino médio, achei esta atividade desafiadora, mas extremamente gratificante. Foi emocionante ver como os conceitos teóricos que aprendemos em aula podem ser aplicados na prática para criar algo funcional e útil. No entanto, o processo de aprendizado foi gradual, e algumas partes da atividade, especialmente o uso de React e Git/GitHub, foram um pouco complicadas no início. No entanto, com a orientação adequada e a prática contínua, fui capaz de superar esses obstáculos e entender melhor essas tecnologias. No geral, sinto que esta atividade foi uma excelente oportunidade para desenvolver minhas habilidades de programação e me preparar para projetos mais avançados no futuro. Além disso, a experiência de colaborar com meus colegas de classe no GitHub também foi muito enriquecedora e me deu uma visão do trabalho em equipe no desenvolvimento de software.
