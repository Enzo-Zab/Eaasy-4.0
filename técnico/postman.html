Meus pensamentos sobre esta atividade:

Habilidades Desenvolvidas: Habilidades Desenvolvidas: Compreensão dos conceitos básicos de programação Java, SQL e desenvolvimento de aplicações web com o NetBeans. Habilidade de programação em Java para criar funcionalidades interativas, manipulação de banco de dados e criação de API REST. Aprendizado sobre a estruturação e o design de uma interface de usuário utilizando o NetBeans. Introdução ao desenvolvimento de operações CRUD (Create, Read, Update, Delete) para manipulação de dados em um sistema.

Objetivo da Atividade: O objetivo principal da atividade era desenvolver um sistema CRUD (Create, Read, Update, Delete) de veículos utilizando a IDE NetBeans, Java, e integração com um banco de dados. A atividade visava ajudar os alunos a consolidar seus conhecimentos de programação orientada a objetos, manipulação de banco de dados e desenvolvimento de aplicações web. A atividade também proporcionou uma introdução prática ao desenvolvimento de APIs REST e ao uso do Postman para testes.
Assunto: O assunto da atividade foi a criação de um sistema de gerenciamento de veículos, onde os alunos deveriam implementar as operações CRUD para adicionar, visualizar, atualizar e excluir registros de veículos. A atividade envolveu a aplicação de conceitos de programação orientada a objetos para modelar e manipular os dados dos veículos, bem como a criação de uma interface de usuário simples e funcional no NetBeans. Além disso, os alunos aprenderam a criar uma API REST para interação com o banco de dados e testaram suas funcionalidades usando o Postman.

Crítica Pessoal: Como aluno do ensino médio, achei esta atividade desafiadora, porém muito enriquecedora. Foi interessante ver como os conceitos teóricos de programação podem ser aplicados na prática para criar um sistema funcional e útil. No entanto, o processo de aprendizado foi gradual, e algumas partes da atividade, especialmente a integração com o banco de dados e a criação da API REST, foram um pouco complicadas no início. Com a orientação adequada e a prática contínua, consegui superar esses obstáculos e entender melhor essas tecnologias. No geral, sinto que esta atividade foi uma excelente oportunidade para desenvolver minhas habilidades de programação, compreender a importância da manipulação de dados em aplicações reais e me preparar para projetos mais avançados no futuro. Além disso, a experiência de testar e colaborar com meus colegas de classe usando o Postman foi muito enriquecedora e me deu uma visão do trabalho em equipe no desenvolvimento de software.
