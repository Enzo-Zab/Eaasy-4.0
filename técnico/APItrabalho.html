Habilidades Desenvolvidas:

Durante o trabalho com APIs, desenvolvi habilidades cruciais em design, desenvolvimento e integração de interfaces de programação.
Aprendi a projetar e implementar APIs RESTful, garantindo que elas fossem robustas e seguras. Tornei-me proficiente no uso de
ferramentas para testar e documentar APIs, como Postman e Swagger, o que facilitou a comunicação com outras equipes e garantiu
que as APIs fossem bem compreendidas e utilizadas. Além disso, refinei minhas habilidades em lidar com autenticação e autorização,
melhorando a segurança das APIs e a gestão de acesso a dados sensíveis.

Objetivo da Atividade:
O objetivo principal do trabalho com APIs foi criar e gerenciar interfaces que permitissem a comunicação eficiente entre diferentes
sistemas e serviços. A atividade visava desenvolver APIs que fossem fáceis de integrar e utilizar, melhorando a interoperabilidade
e a funcionalidade dos aplicativos. A meta era garantir que as APIs atendesse às necessidades dos usuários e integradores,
oferecendo uma base sólida para o desenvolvimento de novas funcionalidades e serviços.

Assunto: O assunto do trabalho com APIs incluiu os seguintes componentes:
Design e Estruturação: Criação da arquitetura da API, definindo endpoints, métodos HTTP e estruturas de dados para garantir uma interface intuitiva e eficiente.
Implementação: Desenvolvimento das APIs com foco em desempenho e escalabilidade, utilizando frameworks e bibliotecas adequadas para garantir a robustez e a confiabilidade do serviço.
Autenticação e Autorização: Implementação de mecanismos de segurança, como OAuth e JWT, para proteger a API e controlar o acesso aos recursos e dados.
Documentação: Utilização de ferramentas como Swagger para criar uma documentação clara e acessível, que facilitasse a compreensão e a integração da API por outros desenvolvedores.
Testes e Validação: Realização de testes rigorosos para verificar a funcionalidade, a segurança e o desempenho da API, garantindo que ela atendesse aos requisitos e funcionasse conforme o esperado.
Crítica Pessoal: Trabalhar com APIs foi uma experiência extremamente enriquecedora e desafiadora. O projeto me permitiu aprimorar minhas habilidades em design e desenvolvimento de interfaces, e a experiência prática foi fundamental para entender os requisitos e as melhores práticas para criar APIs eficientes e seguras. Um dos maiores desafios foi garantir que a API fosse suficientemente flexível para se adaptar às mudanças e extensões futuras, enquanto mantinha a integridade e a segurança. Isso exigiu uma abordagem cuidadosa e um entendimento profundo dos requisitos dos usuários e das melhores práticas de desenvolvimento. No geral, o trabalho com APIs me proporcionou uma visão mais abrangente sobre como integrar sistemas e serviços de forma eficaz e preparou-me melhor para enfrentar desafios relacionados ao desenvolvimento de interfaces e à comunicação entre diferentes componentes de software.
